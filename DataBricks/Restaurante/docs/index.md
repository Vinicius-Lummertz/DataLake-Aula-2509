# Bem-vindo à Documentação do Pipeline Lakehouse

Este site contém a documentação técnica para o pipeline de dados de análise de vendas de restaurantes, implementado no Databricks.

O projeto demonstra a implementação da **Arquitetura Medalhão** (Bronze, Silver, Gold) para transformar dados brutos em insights de negócios.

## Componentes Principais

* **[Arquitetura Medalhão](arquitetura.md)**: Entenda a lógica de separação das camadas `bronze`, `silver` e `gold`.
* **[Modelo de Dados (Gold)](modelo_dados_gold.md)**: Explore o Star Schema final, pronto para BI.
* **[Detalhes dos Notebooks](notebooks/index.md)**: Veja o passo a passo de cada script do pipeline.
